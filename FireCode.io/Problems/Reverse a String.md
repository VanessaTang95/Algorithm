## Description
> Write a method that takes in a String and returns the reversed version of the String.

## Example
```
reverseString("abcde") -> "edcba"
reverseString("1") -> "1"
reverseString("") -> ""
reverse("madam") -> "madam"
reverse(null) -> null
```
